Step by Step Procedure in PC Assembly/Disassembly

Computer Assembly is the process of putting all the internal components of a computer system to make the computer function. Computer Disassembly is the process of disassembling the computure into its parts.

Steps:

  1. Prepare the tools needed for the task
    • Before starting the disassembly procedure make sure the computer you are using is functional. Make sure you have the correct and complete tools and equipment:
      • Screwdriver (for slotted and Phillips head screws)
      • Adjustable wrench
      • Manuals
      • Small Container to hold screws
      • Long-nosed Pliers
      • Thermal Paste
      • Small Flashlight (if needed)
      • Anti-static Wrist Wrap
  1. Open the CPU Case side panel covers
    • Open the CPU case by removing the side panels. Locate the screws that holds the side panel cover using the screw drivers and gently remove the cover.
  1. Wear the anti-static wrist wrap to ground yourself
    • Wear the anti-static wrist wrap and connect the other end to the CPU case if wrist wrap has no hook or clip, just make sure the other end is wedge to any metal nearby. This will prevent the build-up static electricity from your body from damaging the sensitive computer components.
  1. Install the Motherboard in the CPU case
    • Follow these steps to install the motherboard in the CPU case:
      • Place the motherboard carefully to the CPU case make sure the motherboard is aligned to the stand-off screws and the I/O bazel is aligned with the back panel connectors.
      • Locate the screw slots in the motherboard and carefully fasten the screws using the screw driver make sure all screws are properly placed.
  1. Installing the Hard Drive to the drive bays
    • To install the drive:
      • Slide the drive into the drive bay until the screw holes are lined up make sure the hard drive is in correctorientation. Refer to the figure below.
      • Install the screws.
  1. Install the CPU (Processor) to the CPU socket
    • To install the CPU in the motherboard:
      • Find the corner marking with an arrow which designates the pin 1 of the processor like the figure shown above. Consult the manufacturer’s documentation for specific information about your processor.
      • Lift the small metal rod next to the socket.
      • Find the corresponding marking on the CPU socket and insert the CPU so that the markings are lined up.
      • Push the rod down to properly lock the CPU.
  1. Installing the memory modules (RAM)
    • To install the memory modules:
      • Insert the RAM in the memory slots make sure the notch of the and slots are perfectly aligned
      • Press gently until the both ends are properly inserted in the memory sockets and make sure the tabs on both sides are lock.
  1. Installing the CPU fan:
    • The CPU fan of each computer unit varies on the manufacturer. To view the detailed step for installing the CPU fan you should visit the manufacturer’s website or read the hardware manual.
      • Place thermal paste on the top of the CPU ( if needed).
      • Carefully aligned the CPU assembly to the CPU and mount screws or lock to keep the CPU Fan in place.
  1. Install the power supply unit
    • To install the power supply:
      • Align the mounting holes in the case and power supply.
      • Insert screws and tighten.
  1. Connect the internal cables
    • Before finalizing the connections, make sure all the components are properly connected and secured. You may refer the motherboard manual to carefully check the connections inside the motherboard. There are two kinds of connections, power and data.
      • Install the 24 pins power connector from the power supply to the motherboard. Connect also the power cable for the CPU fan, hard drive and Optical Drive.
      • Refer to the motherboard manual for the connections of the USB, speaker and front panel devices.
      • Tie cables if necessary to organize the cables for ease of access in the troubleshooting process.
  1. Checking and Testing
    • Before proceeding to the testing stage, make sure all the necessary connections are checked specially the power connections. Turn-on the computer and check if the system unit is functioning the way it should be.
